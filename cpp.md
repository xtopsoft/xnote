C++ 扩展名
--------------------------------------------------------------------------------
|C++实现	|源代码文件的扩展名	|头文件扩展名	|
|:--------|:--------|:--------|

|Linux	|C, cc, cxx, cpp, c++	|h, hpp, hxx	|
|VC++	|cpp, cxx, cc	|h, hpp, hxx	|

C++ 指定源文件编码
--------------------------------------------------------------------------------
- g++ -finput-charset=UTF-16 hello.cc
- vc 仅仅支持ascii码

C/C++在函数参数上的区别
--------------------------------------------------------------------------------
###C
> int func() 保存沉默

###C++
> int func() 表示 int func(void)

--------------------------------------------------------------------------------

C++的做法是尽可能在首次使用变量前声明它.
